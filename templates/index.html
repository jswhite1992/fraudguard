What is FraudGuard?

FraudGuard is an application that uses machine learning models to predict fraudulent transactions based on CSV files. Simply upload your CSV file, and FraudGuard will analyze it and provide predictions on the likelihood of fraud for each transaction.

Models Used

FraudGuard utilizes the following machine learning models:

  • Logistic Regression
  • Random Forest
  • Support Vector Machine (SVM)
  • XGBoost

Code Explanation

The Flask application handles the file upload and processing. Upon selecting a CSV file, the application will display the progress bar while processing the predictions using the pre-trained models. The predictions are then displayed on the results page.
